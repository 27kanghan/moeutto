## 세부 사항 기획

----

### 메인 기능

- 회원
    - 소셜 로그인 예정 (카카오)
    - 회원 가입 / 로그인
    - 마이페이지
        - 내 옷장 공개 여부
    - 회원 탈퇴
- 옷장 관리
    - 옷 등록
        - 사진 여러 장 가능
        - 카테고리
            - 상의, 하의 => 자세하게 분류
        - 추가 정보
            - 계절, 두께 등
    - 옷 수정 및 삭제
    - 옷 목록
        - 정렬 기능
    - 옷 상세 보기
- 옷 추천
    - 날씨 등 옷장에 등록된 정보를 기반으로 추천
    - 추천도는 팀에서 생성하여 이를 적용할 예정
