# 추천을 위한 크롤링
1. cron을 이용한 주기적인 크롤링
2. Vision Transformer를 통한 패션 feature 추출
3. Vector store에 저장

## 사용법
### 설치
  - 모듈 설치
    - pip install -r requirements.txt
  - 실행 
    - python main.py
## 주의사항
- fashion-clip img encoder를 사용하여 관련 라이브러리 설치가 필요합니다
  - install.ipynb를 실행하여 설치해주세요
- python version : 3.10
