# 추천을 위한 크롤링
1. cron을 이용한 주기적인 크롤링
2. fashion clip Image encoder을 적용한 임베딩
3. ChromaDB에 임베딩된 이미지와 이름, 가격, 이미지url이 포함된 메타데이터 저장

## 사용법
### 설치
  - 모듈 설치
    - pip install -r requirements.txt
  - 실행 
    - python main.py
## 주의사항
- python version : 3.10
